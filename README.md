# Credit Decision Data Platform

## Overview

This repository contains the solution for the Credit & Lending Data Engineering Assessment for MAL AI-Native Digital Bank.

The proposed architecture implements a cloud-native, production-ready data platform using AWS services and follows the Medallion (Bronze, Silver, Gold) architecture to support secure, scalable, and auditable credit decisioning.

## Solution Highlights

- Multi-source data ingestion
- Medallion Architecture (Bronze / Silver / Gold)
- Customer Identity Resolution
- Conflict Resolution Logic
- Immutable Decision Traceability
- Data Quality Scorecards
- Portfolio Monitoring Mart
- AWS-based scalable architecture

## Technology Stack

- Python
- SQL
- AWS S3
- AWS Glue
- Apache Airflow
- PostgreSQL
